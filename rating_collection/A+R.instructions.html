1. Attractiveness of the Document Summary

In this task you are given a question and a document summary. You will be asked to judge if examining the full document (from which the summary was extracted) is useful for answering the question.

Task (Attractiveness):

Do you think examining the full document will be useful to answer the question?
HINT: Do not try to follow any links while answering this question, but think if following the link would be worthy.

Possible Answers (Attractiveness):

  • Likely, yes should be chosen when the summary makes you believe the document is worth examining.
  • Likely, no should be chosen when the summary makes you believe the document is not interesting.
  • Question/summary is not in English should be chosen only if you are sure that the question and/or summary is in language that you don't understand. Keep in mind that it may be just a technical term in English, so do not select this answer unless you are absolutely sure.

2. Full Document Relevance

Once you are done with the Attractiveness task, you may proceed to the next one. In this task, you will be given a question and a web document to judge. You will be asked to judge if the document contains an answer to the question or not. Your rating should be made based on our rules only; this task is not asking for opinion.

Task (Relevance)

Does the document contain an answer to the question?
HINT: If you are familiar with document relevance tasks, this is one is very similar.

Possible Answers (Relevance):

  • Perfect match is for documents that match the question perfectly. E.g., the Wikipedia page for the question [genome] or the Facebook homepage for the question [facebook].
  • Relevant document with possible differences when the document matches the question in general but there is ambiguity that does not allow to choose "Perfect match" with certainty. Also choose this variant if there are multiple interpretation of the question and the document matches to only one of them (for example, the question [jaguar speed] may refer to the car or the animal).
  • Marginally relevant: obvious differences should be chosen when there is a logical connection between the question and the document, but there is also an obvious mismatch. For example, the question [GDP of Switzerland] is related to the page talking about languages of Switzerland, but those are different things.
  • Irrelevant / Offtopic is for cases when the question and the document are talking about different things, even though there might be some common words. E.g., a recipe of Peking duck is irrelevant for the question [peking university].
  • Cannot open the document should only be chosen if you cannot open the document or get an error.
  • The document is not in English should be chosen only if you are sure that the document is in language you don't understand. Keep in mind that it may be a technical term in English, so do not select this answer unless you are absolutely sure.
